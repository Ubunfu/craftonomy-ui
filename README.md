## Configuration

### Environment Variables
| Parameter                   | Description                                                                     | Default | Required? |
|-----------------------------|---------------------------------------------------------------------------------|---------|-----------|
| VUE_APP_MC_SHOP_CATALOG_URL | URL for the web service returning the mc-shop item catalog.                     | n/a     | Yes       |
